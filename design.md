# トイレの冒険 - インターフェース設計書

| トークン | ライトモード | ダークモード | 用途 |
| :--- | :--- | :--- | :--- |
| primary | #FF8C42 (オレンジ) | #FFB07C | メインアクセント、ボタン |
| background | #FFF8F0 (クリーム) | #1A1410 | 画面背景 |
| surface | #FFFFFF | #2A2420 | カード背景 |

## スクリーン一覧

### タブ構成（ボトムタブ4つ）

| タブ | アイコン | 画面名 | 説明 |
| :--- | :--- | :--- | :--- |
| ホーム | 🏠 house.fill | HomeScreen | トイレ記録のメイン画面 |
| シール帳 | ⭐ star.fill | StickerBookScreen | シール帳＆ご褒美確認 |
| タイムライン | 💬 bubble.left.and.bubble.right.fill | TimelineScreen | ソーシャルタイムライン |
| プロフィール | 👤 person.fill | ProfileScreen | ユーザー設定 |

### 画面詳細

**1. ホーム画面 (HomeScreen)**
- 上部: アバター表示（現在の着せ替え状態）＋ニックネーム＋「今日のがんばり」カウンター
- 中央: 3つの大きな記録ボタン（縦並び）
  - 「🚽 べんざにすわれた！」（ライトブルー背景）
  - 「💧 おしっこできた！」（ライトイエロー背景）
  - 「💩 うんちできた！」（ライトピンク背景）
- 下部: 今日の記録サマリー（アイコン＋回数）

**2. シール帳画面 (StickerBookScreen)**
- 上部: 現在のシール枚数 / 次のご褒美までの残り枚数（プログレスバー）
- 中央: シール帳グリッド（5列×複数行）。獲得済みシールが貼られ、未獲得スロットは薄いグレーの丸で表示
- 下部: 「もらったアイテム」ボタン → 獲得済みアバターアイテム一覧へ遷移

**3. タイムライン画面 (TimelineScreen)**
- 上部: 「みんなのがんばり」タイトル
- リスト: カード形式で時系列表示
  - 各カード: アバターアイコン＋ニックネーム＋「うんちできた！」等のメッセージ＋時刻
  - 応援スタンプボタン（「すごい！👏」「がんばったね！💪」「やったね！🎉」）
  - スタンプ数表示

**4. プロフィール画面 (ProfileScreen)**
- アバター表示（大きめ）＋着せ替えボタン
- 設定フォーム: ニックネーム、性別選択、年齢（月齢）
- タイムライン投稿ON/OFF設定
- アプリ情報

### モーダル・オーバーレイ

**初回セットアップ画面 (OnboardingScreen)**
- ステップ1: ニックネーム入力
- ステップ2: 性別選択
- ステップ3: 年齢（月齢）設定
- ステップ4: アバター選択（6種類程度）

**アバター着せ替えモーダル (AvatarDressUpModal)**
- 上部: アバタープレビュー（現在の着せ替え状態）
- 下部: 獲得済みアイテムのカテゴリ別タブ（帽子、服、アクセサリー等）
- アイテムタップで即時プレビュー＋適用

**ご褒美獲得アニメーション (RewardCelebration)**
- フルスクリーンオーバーレイ
- 紙吹雪アニメーション＋新アイテムの表示
- 「やったね！」メッセージ

## 主要ユーザーフロー

**フロー1: 初回起動**
アプリ起動 → スプラッシュ → オンボーディング（ニックネーム→性別→年齢→アバター選択）→ ホーム画面

**フロー2: トイレ記録**
ホーム画面 → 記録ボタンタップ → シール獲得アニメーション → シール帳に反映 → （シール帳が一杯の場合）ご褒美獲得アニメーション → アバターアイテム付与

**フロー3: タイムライン閲覧＆応援**
タイムラインタブ → 投稿一覧スクロール → 応援スタンプタップ → スタンプ数更新

**フロー4: アバター着せ替え**
プロフィール画面 → 着せ替えボタン → アバター着せ替えモーダル → アイテム選択 → 適用 → 保存
